Release Notes
=============
Initial Release v1.0
---------------------
- LM2596S-5.0 switching regulator.
- DS1307 RTC with CR1220 backup battery.
- Shift register, BT1308W triac solenoid driver.
- SMBJ48CA bidirectional TVS.
- 5V mini-relay (contact rating 120VAC/2A).
- 2A solder-on fuse.
